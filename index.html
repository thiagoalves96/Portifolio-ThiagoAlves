Projeto de CRUD em Java com Servlet

Este projeto é um sistema de gerenciamento de funcionários em Java com Servlet, conectando-se a um banco de dados MySQL. Ele oferece operações básicas de CRUD (Create, Read, Update, Delete) e é licenciado sob a MIT License.

CRUD em Java Swing

Este projeto visa criar um sistema de autenticação avançado em Java Swing, integrado com um banco de dados, para login e logout. Recursos incluem tela de login, verificação de credenciais, tela principal após login, registro de usuário e recuperação de senha. Utiliza Java para lógica do programa, Swing para interface gráfica e um banco de dados (como MySQL ou PostgreSQL) para armazenar informações de usuário. Benefícios incluem um sistema robusto e seguro, oportunidade de aprender Java Swing com banco de dados e segurança avançada. Os objetivos são desenvolver um sistema eficiente e seguro, proporcionar uma boa experiência ao usuário e aprofundar conhecimentos em Java, Swing e banco de dados.

Lógica de Programação

Este projeto concluído é o Jogo da Sorte, desenvolvido em Java. Os jogadores tentam adivinhar um número escolhido aleatoriamente pela máquina dentro de um intervalo definido, recebendo dicas para orientar suas suposições. Utiliza Java (Versão 8) e Eclipse IDE. Os jogadores têm três tentativas para acertar o número sorteado. Este projeto está licenciado sob a MIT License e aceita contribuições.
